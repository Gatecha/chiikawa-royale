# Chiikawa Park Bomb Battle

A local browser game inspired by classic Bomberman-style arena play, rebuilt with a cute Chiikawa park theme.

## Play

Open `index.html` in a browser.

## Controls

- Player 1, Hachiware: WASD or Arrow keys to move, Space/E/F to place a bomb
- Player 2, Usagi: IJKL to move, Enter to place a bomb
- CPU players: Momonga and Shisa

## Gameplay

Break crates, grab powerups, chain bomb explosions, and be the last friend standing.

Powerups:

- `F`: longer flame range
- `B`: extra active bomb
- `S`: faster movement
